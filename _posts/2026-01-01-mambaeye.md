We propose **MambaEye**, a causal sequential visual encoder that achieves input-size agnosticism using a pure Mamba2 backbone. By employing a strictly unidirectional approach and relative move embeddings, MambaEye ensures translation invariance and adaptability to arbitrary resolutions. A novel diffusion-inspired loss provides dense supervision, enabling the model to build confidence sequentially. MambaEye demonstrates robust performance on ImageNet-1K, scaling to high resolutions (e.g., 1536x1536) while maintaining linear time and memory complexity.
